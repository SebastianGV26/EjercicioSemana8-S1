Respuestas al ejercicio - Semana 8

¿Qué es un algoritmo?
Es la sucesión de pasos para darle solución a un tema.
Datos de entrada / salida
Los datos de entrada son las variables que reciben la información, ya sea del usuario o del request. Mientras que los datos de salida son las variables que muestran la respuesta por lado del sistema
¿Que es JavaScript?
Es el lenguaje de programación mediante el cual podemos recibir respuestas por parte del servidor y también hacer la web más interactiva, esto con el fin de brindarle una mejor experiencia al usuario
¿Que es Es5 y JS moderno?
El Es5, es el estandar antiguo y compatible con los navegadores, donde solo se podian definir las variables con la palabra "var". Mientras que en JS moderno, se pueden definir con "let" o "const" haciendo que el código sea más concreto y sin errores.
Que son los tipos de datos
Los tipos de datos son las clasificaciones que les damos a las variables según sea la necesidad que tengamos.
Tipos de datos
String, Number, boolean, null, undefined, entre otros.
Diferencias entre let, var y const
Las variables "let" pueden ser modificadas, pero no declaradas nuevamente, las variables "const" no pueden ser modificadas, ni declaradas nuevamente, ya que representan un valor constante y la variables "var" puede ser modificadas y declaradas nuevamente dentro de su ámbito.
